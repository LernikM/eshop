# URM CLient

The documentation of the URM Client

## Running development server

Run "npm start" for a dev server.
Navigate to http://localhost:8080 .
The app will automatically reload if you change any of the source files.

## Running eslint

Run "npm run pretest"

## Running tests

Run "npm test" to execute the tests via JEST with jest-teamcity-reporter.
Run "npm run local-test" to execute the tests via JEST without jest-teamcity-reporter.

## Running build

Run "npm run build" to build the project. The build artifacts will be stored in the build/ directory.
